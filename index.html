CHANAKYA EMR API DOCUMENTS



Service -01 Currency Converter


1. Set/Update currency details

Request Parameters

  • currency1
  • currency2
  • currency1_value

Description: In this service can insert new data or update existing data and gives JSON response

If data is present updates if not inserting as new value

End point: http://....../setCurrencyValues? (Post method)

2. Convert currency

Request Parameters

  • currency1
  • currency2
  • currency1_value

Description: In this service it will convert the user entered currency1 amount with the currency1_value and gives currency2 value as a JSON response

End point: http://....../getCurrencyValue? (Post method)

3. Convert all currency units present in database

Request Parameters

  • currency1
  • currency1_value

Description: This service convert user entered currency1 amount with the all currency1_value present in the database and gives currency2 value as a JSON response.

End point: http://....../getAllCurrencyValue? (Post method)

4. Get all currency details present in database

Request Parameters

  • No parameter

Description: This service gives list of all currency details.

End point: http://....../getAllCurrecyDetails? (Get method)

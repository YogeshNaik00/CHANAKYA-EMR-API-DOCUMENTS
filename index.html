CHANAKYA EMR API DOCUMENTS



Service -01 Currency Converter


1. Set currency details - http://....../setCurrencyValues?

Request: {String, String, float} (currency1, currency2, currency1_value)

Response: { "status": "Success" }

2. Convert currency - http://....../getCurrencyValue?

Request: {String, String, float} (currency1, currency2, currency1_value)

Response: { currency1: "INR", currency2: "USD", outputCurrencyValue: 1.2456747, currencyUnit: "USD" }

3. Convert all currency units present in database - http://....../getAllCurrencyValue?

Request {String, float} (currency1, currency1_value)

Response: [ { currency1: "INR", currency2: "USD", outputCurrencyValue: 0.9688581, currencyUnit: "USD" }, { currency1: "INR", currency2: "EUR", outputCurrencyValue: 0.81065434, currencyUnit: "EUR" }, { currency1: "INR", currency2: "YEN", outputCurrencyValue: 189.1892, currencyUnit: "YEN" } ]

4. Get all currency details present in database - http://....../getAllCurrencyDetails?

Request - no parameter

Response: [ { id: 1, currency1: "INR", currency2: "USD", currency1_value: 72.25 }, { id: 2, currency1: "INR", currency2: "EUR", currency1_value: 86.35 }, { id: 3, currency1: "INR", currency2: "YEN", currency1_value: 0.37 } ]
